          Returns Promise<void>

          A Promise that resolves when the command execution is complete.

          Remarks

          This function executes a command by calling its handler function with the provided arguments. It also updates the execution history and handles the output of the command.

          Example

          const command: Command = {
            name: 'example',
            handler: async (args, commandValue) => {
              // Command handler logic
            },
          };

          await executeCommand('example arg1 arg2', command);

          Remarks

          This function searches for a command that matches the provided text and returns its corresponding handler function. If no matching command is found, it falls back to the default handler or the fallback handler.

          Example
