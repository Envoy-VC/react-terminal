# useTerminal Hook

The Library exposes a `useTerminal` hook which can be used to control the terminal. The hook returns an object with the following properties:

- **inputRef**: `React.RefObject<HTMLTextAreaElement>` - A reference to the input box.
- **terminalRef**: `React.RefObject<HTMLDivElement>` - A reference to the terminal Container.
- **theme**: `Theme` - The current theme of the terminal.
- **fontSize**: `number` - The current font size of the terminal.
- **inputBoxValue**: `string` - The current value of the input box.
- **focus**: `() => void` - A function to focus the input box.
- **blur**: `() => void` - A function to blur the input box.
- **scrollToTop**: `() => void` - A function to scroll to the top of the terminal.
- **scrollToBottom**: `() => void` - A function to scroll to the bottom of the terminal.
- **scrollInView**: `(index: number) => void` - A function to scroll a specific line into view.
- **writeToTerminal**: `(text: string) => void` - A function to write text to the terminal.
- **clearTerminal**: `() => void` - A function to clear the terminal.
- **setFontSize**: `(size: number) => void` - A function to set the font size of the terminal.
- **setTheme**: `(theme: Theme) => void` - A function to set the theme of the terminal.
- **setInputBoxValue**: `(value: string) => void` - A function to set the value of the input box.

Example:

```tsx
import { Terminal, useTerminal } from '@envoy1084/react-terminal';

const MyComponent = () => {
  const { focus, scrollToTop, scrollToBottom, scrollInView, writeToTerminal } =
    useTerminal();

  return (
    <div>
      <Terminal />
      <button onClick={focus}>Focus</button>
      <button onClick={scrollToTop}>Scroll to Top</button>
      <button onClick={scrollToBottom}>Scroll to Bottom</button>
      <button onClick={() => scrollInView({ block: 'nearest' })}>
        Scroll into View
      </button>
      <button onClick={() => writeToTerminal('Hello World!')}>
        Write to Terminal
      </button>
    </div>
  );
};
```
