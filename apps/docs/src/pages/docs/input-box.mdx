# Input Box

You can customize the Input by setting the `inputBox` prop. The InputBoxProps extends the `HTMLTextAreaElement` so you can pass any valid textarea props.

The available options are:

- **prompt**: `React.ReactNode` - The prompt to be displayed before the user input.
- **cursor**: `'underscore' | 'block' | 'bar' | React.ReactNode` - The cursor style to be used in the input box. **Defaults to 'underscore'**.

Example

```tsx
import { Terminal } from '@envoy1084/react-terminal';

const customPrompt = <span className='!text-blue-300'>{`>>>`}</span>;

const App = () => {
  return (
    <Terminal
      inputBox={{
        prompt: customPrompt,
        cursor: 'block',
      }}
    />
  );
};

export default App;
```
