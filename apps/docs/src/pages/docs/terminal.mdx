# Terminal

The Terminal component is used to render the terminal.

It takes the following props:

- **[theme](/docs/theme)**: `Theme` - The theme for the terminal.
- **[welcomeMessage](/docs/termina)**: `React.ReactNode` - The welcome message for the terminal.
- **[showWelcomeMessage](/docs/terminal)**: `boolean` - whether to show the welcome message or not. _defaults to true_
- **[fontSize](/docs/terminal)**: `number` - The font size for the terminal. _defaults to 16_
- **[showTitleBar](/docs/terminal)**: `boolean` - Determines whether to show the title bar or not. _defaults to true_
- **[titleBar](/docs/title-bar)**: `TitleBarProps` - The props for the title bar component. _defaults to "React Terminal"_
- **[titleBarProps](/docs/terminal)**: `React.ComponentPropsWithoutRef<'div'>` - The props for the title bar container.
- **[inputBox](/docs/input-box)**: `InputBoxProps` : The props for the input box component.
- **[inputBoxProps](/docs/terminal)**: `React.ComponentPropsWithoutRef<'textarea'>` - The props for the textarea container.
- **[executingLoader](/docs/terminal)**: `React.ReactNode` - The React node to be rendered while a command is executing.
- **[commands](/docs/commands)**: `Command[]` - An array of commands available in the terminal. _defaults to []_
- **[enableDefaultCommands](/docs/terminal)**: `boolean` - Determines whether to enable the default commands or not. _defaults to true_
- **[defaultHandler](/docs/commands)**: `Command` - The default command handler.
- **[htmlRenderer](/docs/html-renderer)**: `HTMLRenderer` - The HTML renderer for the terminal.

Example:

```tsx
import { Terminal, themes } from '@envoy1084/react-terminal';

const App = () => {
  return (
    <Terminal
      theme={themes.aurora_x}
      fontSize={16}
      showTitleBar={true}
      titleBar={{
        header: 'Terminal',
      }}
      inputBox={{
        cursor: 'block',
      }}
      executingLoader={<div>Loading...</div>}
    />
  );
};
```
