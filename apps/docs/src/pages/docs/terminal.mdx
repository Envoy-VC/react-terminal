# Terminal

The Terminal component is used to render the terminal.

It takes the following props:

- **[theme](/docs/theme)**: `Theme?` - The theme for the terminal.
- **[fontSize]()**: `number?` - The font size for the terminal. _defaults to 16_
- **[autoScroll]()**: `boolean?` - Determines whether to auto scroll the terminal or not. _defaults to true_
- **[commands](/docs/commands)**: `Command[]?` - An array of commands available in the terminal. _defaults to []_
- **[enableDefaultCommands]()**: `boolean?` - Determines whether to enable the default commands or not. _defaults to true_
- **[defaultHandler](/docs/commands)**: `Command?` - The default command handler.

Example:

```tsx
import { Terminal, themes } from '@envoy1084/react-terminal';

const MyComponent = () => {
  return (
    <Terminal theme={themes.aurora_x} fontSize={18} commands={[]}>
      {/** Other Terminal Components */}
    </Terminal>
  );
};

export default MyComponent;
```
